# Контрольные вопросы к лабораторной работе №1

## 1. Что такое «командная оболочка» / «командный интерпретатор»?
**Ответ:** Это программа, которая обеспечивает интерфейс для взаимодействия пользователя с операционной системой. Она принимает команды, введенные пользователем с клавиатуры, и преобразует их в форму, понятную ядру ОС для выполнения [citation:3][citation:4].

## 2. Как называется командный интерпретатор и какая его версия в используемой операционной системе?
**Ответ:** В ОС Windows основными интерпретаторами являются `Cmd.exe` (Command Prompt) и `PowerShell`. Для `Cmd.exe` версию можно узнать командой `ver`. Версия `PowerShell` определяется командой `$PSVersionTable.PSVersion`.

## 3. Для каких задач предназначено приложение Terminal в macOS?
**Ответ:** Terminal в macOS предназначен для доступа к командной строке Unix-подобной операционной системы. Он используется для выполнения широкого спектра задач: управления файлами и каталогами, настройки системы, установки и запуска программ, написания скриптов и администрирования серверов [citation:5].

## 4. Приведите полный путь к домашней папке пользователя, под которым вы авторизованы в системе. Привести пример команды, позволяющей определить путь к домашнему (текущему) каталогу в консоли.
**Ответ:** В Windows домашняя папка обычно находится по пути `C:\Users\Имя_Пользователя`. Определить её можно командой `echo %USERPROFILE%`. Текущий каталог можно узнать командой `cd` (без параметров) или `chdir`.

## 5. Какой командой можем вывести содержимое каталога системных конфигурационных файлов?
**Ответ:** В Windows для вывода содержимого каталога (например, `C:\Windows\System32\drivers\etc`) используется команда `dir C:\Windows\System32\drivers\etc`. В Unix-подобных системах (Linux/macOS) для каталога `/etc` используется команда `ls /etc`.
